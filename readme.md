## About Forkly

Forkly is a full-stack food delivery web application built as a SaaS platform, 
enabling users to browse restaurants, explore menus, and place orders through 
a clean, responsive interface.

### Tech Stack
- **Frontend:** React (Vite), mobile-first responsive CSS
- **Backend:** Django REST Framework
- **Architecture:** Component-based UI with RESTful API integration

### Features
- 📱 Mobile-first, fully responsive design (built with breakpoint-based CSS architecture)
- 🍔 Browse restaurants and food items with a dynamic, component-driven UI
- 🔗 REST API-powered backend for handling orders, menus, and data
- ⚡ Built with Vite for fast development and optimized builds
